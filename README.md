## 외부 api 목록

## \***\*기술적 도전\*\***

- React.js, Next.js를 통한 SSG, SSR, SPA 사용
- Redux와 thunk를 통한 상태 관리 로직 구현
- Canvas API를 사용한 차트 구현
- TypeScript, SCSS 사용, 비동기 로직 최적화
- jest, react-testing-library를 사용한 TDD(Test Driven Development)
- playwright를 사용한 E2E testing
